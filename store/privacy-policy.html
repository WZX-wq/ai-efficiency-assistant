AI效率助手 - 隐私政策

最后更新日期：2026年4月13日

概述

AI效率助手（以下简称"本扩展"）非常重视用户隐私。本隐私政策说明了本扩展如何处理信息。

数据收集

本扩展不收集任何用户数据。

  • 我们不收集、存储或传输任何个人信息
  • 我们不使用任何分析工具或追踪技术
  • 我们不记录您的浏览历史或使用习惯

API 调用

本扩展的功能需要调用AI模型API。相关说明如下：

  • 所有API调用均直接从您的浏览器发起，不经过任何中间服务器
  • 您发送到AI API的文本内容仅用于生成AI响应，不会被存储或用于其他目的
  • 具体的隐私政策请参阅您所使用的AI服务提供商的隐私政策

本地存储

本扩展使用Chrome的本地存储功能（chrome.storage.local）保存以下信息：

  • API Key：用于调用AI服务，仅存储在您的本地浏览器中
  • 偏好设置：如模型选择、温度参数等

这些数据仅保存在您的设备上，不会同步到云端或发送给任何第三方。

权限说明

本扩展需要以下Chrome权限：

  • storage：用于在本地保存您的设置和API Key
  • activeTab：用于获取当前标签页的选中文字
  • contextMenus：用于在右键菜单中添加AI操作选项
  • scripting：用于在网页中注入浮动按钮界面

第三方服务

本扩展可能连接到以下第三方AI服务：

  • 白山智算（api.edgefn.net）
  • DeepSeek（api.deepseek.com）
  • 其他兼容OpenAI API格式的服务

使用这些服务时，其各自的隐私政策适用。本扩展不对第三方服务的隐私做法负责。

变更

我们可能会不时更新本隐私政策。更新后的政策将在此页面上发布。
