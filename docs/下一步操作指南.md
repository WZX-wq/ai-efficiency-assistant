# AI 效率助手 — 下一步操作指南

> 以下步骤需要您手动操作（涉及账号登录），我已经为您准备好了所有材料和命令，直接复制执行即可。

---

## 一、推送到 GitHub（约 5 分钟）

### 1.1 创建 GitHub 仓库

1. 打开 https://github.com/new
2. 仓库名称填写：`ai-efficiency-assistant`
3. 描述填写：`AI效率助手 - 选中文字一键AI改写/扩写/翻译/总结的Chrome扩展`
4. **不要勾选** "Initialize with README"
5. 点击 "Create repository"

### 1.2 推送代码

创建仓库后，GitHub 会显示一个地址，替换下面的 `<你的用户名>` 执行：

```bash
cd /workspace/ai-efficiency-assistant
git remote add origin https://github.com/<你的用户名>/ai-efficiency-assistant.git
git push -u origin main
```

### 1.3 添加 Topics 标签

在仓库页面点击 ⚙️ 齿轮图标，添加以下标签：
`chrome-extension`, `ai`, `productivity`, `deepseek`, `react`, `typescript`

---

## 二、部署到 Vercel（约 3 分钟）

### 2.1 登录 Vercel

1. 打开 https://vercel.com
2. 点击 "Sign Up" → 选择 "Continue with GitHub" → 授权登录

### 2.2 导入项目

1. 登录后点击 "Add New..." → "Project"
2. 在 "Import Git Repository" 中找到 `ai-efficiency-assistant`
3. 点击 "Import"

### 2.3 配置部署

Vercel 会自动检测到配置文件，确认以下设置：
- **Framework Preset**: Vite
- **Root Directory**: `.`（保持默认）
- **Build Command**: `cd web && npm install && npm run build`（自动读取自 vercel.json）
- **Output Directory**: `web/dist`（自动读取自 vercel.json）

点击 "Deploy"，等待 1-2 分钟即可完成。

### 2.4 获取访问地址

部署成功后，Vercel 会分配一个地址，如：
`https://ai-efficiency-assistant.vercel.app`

您可以点击 "Settings" → "Domains" 绑定自定义域名。

---

## 三、发布 Chrome 插件（约 30 分钟 + 审核）

### 3.1 注册开发者账号

1. 打开 https://chrome.google.com/webstore/devconsole
2. 使用 Google 账号登录
3. 支付 **$5 美元** 一次性注册费（信用卡/借记卡）
4. 完成身份验证（需要上传身份证件 + 自拍照）

### 3.2 上传扩展

1. 在 Developer Dashboard 点击 "New Item"
2. 上传 zip 文件：`store/ai-efficiency-assistant.zip`（已为您打包好）
3. 填写商店信息：
   - **名称**：AI效率助手
   - **简短描述**：AI效率助手 - 选中网页文字，一键AI改写、扩写、翻译、总结
   - **详细描述**：复制 `store/store-description.md` 中的内容
   - **分类**：生产力工具 (Productivity)
   - **语言**：中文（简体）、English
4. 上传图标（已有 128x128 图标在 `public/icons/icon-128.png`）
5. 上传截图（建议尺寸 1280x800 或 640x400，至少1张）

### 3.3 隐私政策

在 "Privacy Policy" 栏填写隐私政策 URL。您可以：
- 将 `store/privacy-policy.html` 部署到 GitHub Pages
- 或者直接使用 Vercel 部署后的地址

### 3.4 提交审核

点击 "Submit" 提交审核，通常 **1-3 个工作日** 内完成。

## 五、本地测试（可选）

### 5.1 测试 Chrome 插件

1. 打开 Chrome，访问 `chrome://extensions/`
2. 开启右上角 "开发者模式"
3. 点击 "加载已解压的扩展程序"
4. 选择项目的 `dist/` 目录
5. 点击扩展图标 → 设置中填入白山智算 API Key → 开始使用

### 5.2 测试 Web 前端（本地）

```bash
cd /workspace/ai-efficiency-assistant/web
npm run dev
```

然后打开 http://localhost:3000
