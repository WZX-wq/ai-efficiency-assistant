## 详细描述

## AI 效率助手 - 让AI成为你的超级写作搭档

还在切换窗口复制粘贴到ChatGPT？AI效率助手让你在任意网页上直接调用AI能力！

### 核心功能
- **智能改写**：选中文字一键改写，保持原意但表达更专业
- **一键扩写**：快速扩展内容，补充更多细节和论据
- **多语言翻译**：中英文智能互译，支持更多语言
- **内容总结**：长文精炼摘要，一秒提取核心要点

### 三种使用方式
1. **浮动按钮**：选中文字后自动出现，点击即用
2. **右键菜单**：选中文字右键，选择AI操作
3. **弹出窗口**：点击扩展图标，手动输入文字

### 支持的AI模型
- DeepSeek-V3 / DeepSeek-V3.2
- GLM-5 / GLM-4.7
- Qwen3-235B / MiniMax-M2.5
- 以及更多兼容OpenAI API格式的模型

### 使用方法
1. 安装扩展
2. 点击扩展图标，在设置中填入你的API Key
3. 在任意网页选中文字，选择AI操作即可

### 隐私保护
- 所有API调用直接从你的浏览器发出，不经过第三方服务器
- API Key安全存储在浏览器本地，不会上传
- 不收集任何用户数据

### 适用场景
- 写作优化：改写邮件、文章、报告
- 内容创作：扩写大纲、补充细节
- 阅读辅助：快速翻译外文内容、总结长文
- 学习提升：理解复杂文本、生成学习笔记

## English Description

### AI Efficiency Assistant - Your AI Writing Copilot

Tired of switching tabs to copy-paste into ChatGPT? AI Efficiency Assistant lets you use AI directly on any webpage!

### Features
- **Smart Rewrite**: Rephrase selected text while keeping the original meaning
- **Quick Expand**: Enrich content with more details and arguments
- **Multi-language Translation**: Seamless Chinese-English translation
- **Content Summary**: Extract key points from long articles in seconds

### How to Use
1. Install the extension
2. Click the extension icon and enter your API Key in settings
3. Select text on any webpage and choose an AI action

### Privacy
- All API calls are made directly from your browser
- API Key is stored locally and never uploaded
- No user data is collected
